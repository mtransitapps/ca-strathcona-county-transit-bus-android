Full description:
This app adds Strathcona County Transit buses information to MonTransit.

This app provides the bus schedule.

Strathcona County Transit buses serve Sherwood Park, Strathcona County, Edmonton in Alberta, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by Strathcona County Transit.
https://data.strathcona.ca/

This application is free and open-source:
https://github.com/mtransitapps/ca-strathcona-county-transit-bus-android

This app is not related with Strathcona County Transit.
